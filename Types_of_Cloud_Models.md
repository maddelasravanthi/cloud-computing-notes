# Types of Cloud Models:
There are Four types
1.public
2.private
3.Hybrid
4.Community Cloud

## Public Cloud
Cloud resources are owned and managed by a third-party cloud provider and shared among multiple customers.
Examples:
Microsoft Azure
AWS
Google Cloud

#### Characteristics:
No hardware ownership
Pay-as-you-go
Highly scalable
Internet accessible

#### Advantages:

Low cost,
High scalability,
Easy maintenance,
Global availability

#### Disadvantages:

Less control,Shared infrastructure

## Private Cloud
Cloud infrastructure is dedicated to a single organization.
#### Examples:
On-premise datacenter, Azure Stack, VMware private cloud
#### Characteristics:
Single organization usage , More security control Higher cost
#### Advantages:
High security, Full control, Compliance friendly
#### Disadvantages:
Expensive, Limited scalability, Requires maintenance
## Hybrid Cloud
Combination of Public Cloud + Private Cloud.
#### Characteristics:
Sensitive data stays private, Less sensitive workloads run on public cloud, Flexible architecture
#### Advantages:
Cost efficient ,High flexibility, Better security
#### Disadvantages:

Complex management, Requires integration
## Community Cloud
Cloud infrastructure shared by multiple organizations with similar requirements.

#### Examples:
Government departments, Educational institutions
#### Advantages:
Cost shared, Better compliance

#### Disadvantages:
Limited scalability
Shared control
