# Cloud Service Models

## IaaS (Infrastructure as a Service)
Infrastructure as a Service (IaaS) provides virtualized computing resources-such as servers, storage, and networking-over the internet on a pay-as-you-go basis. It eliminates the need to manage physical hardware, offering flexible scalability for businesses of all sizes.
- Provides virtual machines, storage, networks
- User manages OS and applications
- Example: Azure Virtual Machines

## PaaS (Platform as a Service)
Platform as a Service (PaaS) offers a cloud environment for developing, running, and managing applications without dealing with the complexities of maintaining the underlying infrastructure.
- Provides runtime environment and tools
- Developer focuses only on application
- Example: Azure App Service

## SaaS (Software as a Service)
Software as a Service (SaaS) is the most user-friendly model, providing complete software applications hosted in the cloud
- Fully managed software
- Accessible via browser
- Example: Gmail, Office 365
